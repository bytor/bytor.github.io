Test For Echo
Atlantic PRCD 6853-2 (USA)
September 1996


TRACK LISTING
5.55 Test For Echo 		DESCRIPTION
This is a promo cd single. It has no front insert; the picture-disc cd is a black & white image of a howling wolf, the same image from the Test For Echo album; like all newer releases, the case itself has a Warner logo on the back. This single appears to have been printed in a hurry -- on mine, the black ink from the back insert leaked through the paper to the other side!
